# SmartCart Segmentation System

Machine learning-based customer segmentation system that analyzes shopping behavior and groups customers into meaningful segments for data-driven business insights.

## Project Overview

This project uses machine learning techniques to segment customers based on their shopping behavior. The generated customer groups can help businesses understand purchasing patterns and make data-driven marketing decisions.

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Customer segmentation using clustering techniques
* Visualization of customer groups
* Business insights from segmented customers

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
